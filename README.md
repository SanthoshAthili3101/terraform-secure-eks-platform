# Terraform Secure EKS Platform

Production-grade, multi-environment Amazon EKS platform built with Terraform.

## Features
- Multi-environment (dev / staging / prod)
- Secure-by-default networking
- IAM least privilege + IRSA
- CI/CD-driven Terraform
- FinOps and cost visibility
- Enterprise documentation

## Environments
- dev
- staging
- prod

## Principles
- No manual changes
- Terraform-only infra
- AWS OIDC auth (no static keys)
